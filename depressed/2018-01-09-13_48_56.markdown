For Ubuntu

```shell
apt-get install mutt
apt-get install swaks
apt-get install mailx
apt-get install sharutils
```

For CentOS

```shell
yum install mutt
yum install swaks
yum install mailx
yum install sharutils
```



Usage:

```
mail -s -a user@example.com
```
